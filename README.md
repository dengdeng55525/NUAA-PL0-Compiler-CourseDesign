# NUAA-PL0-Compiler-CourseDesign

南京航空航天大学编译原理课程设计项目，基于 Python 实现 PL/0 小型编译器，并提供命令行与 Flask Web 前端用于查看词法分析、语法分析、语义检查、中间代码、虚拟机执行结果和错误信息。

## 基本信息

- 课程：编译原理课程设计
- 作者：丁俊泽

## 功能概览

- 词法分析：识别 PL/0 关键字、标识符、常量、运算符和界符，并输出 token 序列。
- 语法分析：构建递归下降分析流程，支持错误定位和一定程度的语法恢复。
- 语义检查：处理符号表、作用域、常量/变量/过程声明以及过程调用参数检查。
- 代码生成：生成面向栈式虚拟机的指令序列。
- 优化与执行：包含低风险 peephole 优化，并通过虚拟机解释执行生成的目标代码。
- Web 可视化：展示 tokens、AST、IR、运行输出、错误信息和调试信息。

## 目录说明

- `编译程序文件/`：编译器主体代码、测试脚本和 Web 前端。
- `测试代码/`：用于课程设计验收和功能验证的 PL/0 示例程序。
- `课设报告.pdf`：课程设计报告。
