# 运行方法

1. 安装依赖
   py -3 -m pip install -r web/requirements.txt

2. 启动服务
   py -3 web/app.py

3. 浏览器打开 http://127.0.0.1:5000

## 说明
- 前端页面可以输入 PL/0 源代码，并显示分词、AST、中间代码、运行结果和错误信息。
- 仅当语法/语义无错误时会进行代码生成并自动解释执行（VM）。

## 关键字拼写纠错白名单（可编辑）
- 文件：`web/static/typo_whitelist.txt`
- 修改后刷新网页即可生效（前端会重新拉取该文件）。
- 该功能只在你勾选“关键字拼写纠错（如 the→then）”并点击某条错误的“应用修复”时生效。

## 指令集（假想栈机）
- LIT 0,a   : 常量 a 入栈
- OPR 0,a   : 栈顶运算（a 为子操作码，见下）
- LOD L,a   : 按层差 L/相对地址 a 取变量入栈
- STO L,a   : 栈顶写回变量
- CAL L,a   : 调用过程（入口地址 a，层差 L）
- INT 0,a   : 栈顶指针增加 a（为过程/主程序开辟数据区，含 3 个连接数据）
- JMP 0,a   : 无条件跳转
- JPC 0,a   : 条件跳转（栈顶为 0 时跳转到 a）
- RED L,a   : 读入一个整数写入变量
- WRT 0,0   : 输出栈顶

## OPR 子操作码（本项目自定义但自洽）
- 0  返回/结束
- 1  一元负号
- 2  加法
- 3  减法
- 4  乘法
- 5  除法
- 7  odd
- 8  =
- 9  <>
- 10 <
- 11 >=
- 12 >
- 13 <=

备注：要求.txt 示例提到 A=16 可表示“读入”，本项目将 I/O 设计为独立指令 RED/WRT，因此不使用 OPR=16。
